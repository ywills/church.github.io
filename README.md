# Shady Grove MBC — Multi-page FREE GitHub Pages Site

## Pages
- `index.html` (Home)
- `about.html`
- `word.html` (Daily Word + Daily Inspiration)
- `sermons.html` (Sermon Library)
- `live.html` (Live Bible Study)
- `events.html`
- `give.html`
- `visit.html`
- `contact.html`

## Data files
- `sermons.json` (used by sermons.html)
- `inspiration.json` (used by word.html)

## Assets
- `assets/style.css`
- `assets/app.js` (nav + year + back-to-top)
- `assets/word.js` (Verse of the Day API + daily inspiration rotation)
- `assets/sermons.js` (sermon library loader)

## Publish on GitHub Pages
1. Create a repo named `YOURUSERNAME.github.io`
2. Upload all files/folders in this ZIP into the repo root
3. Settings → Pages → Deploy from branch → main / root
4. Visit https://YOURUSERNAME.github.io

## Live Bible Study
In `live.html`, replace `CHANNEL_ID` with your YouTube Channel ID.
